# Original Project Task

**Date**: 2026-04-24
**Source**: User Request

## Objective
Create a workspace template for AI-assisted projects that ensures:
1.  **Multi-Agent Compatibility**: Works with Gemini, Claude, Codex, etc.
2.  **Traceability**: Records what was done, what needs to be done, goals, and ground truth.
3.  **Separation of Concerns**: Folders for AI (`.agent`), humans (`.human`), sources (`sources`), and tasks (`task`).
4.  **Work Isolation**: Actual project work resides in a dedicated subfolder (e.g., `work/dual_arm_robot`).
5.  **Git Integration**: Proper tracking of prompts, AI actions, and codebase with a tailored `.gitignore`.

## Requirements
- Detailed proposal and implementation.
- Tracking of prompts and AI-made changes.
- Simple, easy-to-follow sub-tasks.
- Ground truth preservation.
